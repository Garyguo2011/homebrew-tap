# Homebrew Tap

Homebrew tap for personal CLI tools.

## Installation

```bash
# Add the tap
brew tap Garyguo2011/tap git@github.com:Garyguo2011/homebrew-tap.git

# Install ot
brew install Garyguo2011/tap/ot
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `ot` | Unofficial command-line interface for Otter.ai |

## Updating

```bash
brew update
brew upgrade ot
```

## Uninstalling

```bash
brew uninstall ot
brew untap Garyguo2011/tap
```
